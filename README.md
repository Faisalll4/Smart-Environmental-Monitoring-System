# Smart Environmental Monitoring System (SEMS)

## Project Overview
Real-time IoT-based environmental monitoring system built using Arduino UNO.

## Name
- Shaikh Faisal 

## College
MGM University, Aurangabad
B.Sc. (Honours) Computer Science — IoT Specialization

## Hardware Used
- Arduino UNO — Central microcontroller
- DHT22 — Temperature & Humidity sensor
- MQ-135 — Air Quality / Gas sensor
- Rain Sensor — Rainfall detection
- LDR — Light intensity sensor
- 16x2 LCD Display (I2C)
- ESP8266 Wi-Fi Module
- Buzzer — Alert system

## Features
- Real-time monitoring of temperature, humidity, air quality, rainfall
- Cloud data upload via ThingSpeak / Blynk
- Threshold-based buzzer alerts
- LCD + mobile app dashboard

## Total Cost
Built under Rs. 1,326 using low-cost components
